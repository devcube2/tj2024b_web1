[ 입출력 ]
	- 입력 : 자료가 밖에서(키보드,마우스)에서 컴퓨터안으로
	- 출력 : 자료가 컴퓨터에서 (모니터,파일,프린터 등등)밖으로

[ 파일 ]
	- 컴퓨터가 자료들을 모아놓은 것
	- 윈도우 기준의 파일은 최소 용량은 바이트

[ 스트림 ]
	1. 정의 : 컴퓨터내 데이터/자료 가 다니는 데이터 통로
	2. 데이터 : *바이트* 단위 의 데이터들이 스트림 통로로 데이터를 주고 받는다.
	3. 사용처 : (프로그램-키보드),(프로그램-파일),(프로그램-네트워크) 등등

[ 자바 파일 입출력 ]
	1. 정의 : 자바에서 저장된 메모리(JVM)을 자바 외부인 파일(윈도우)에 입출력
	2. 목적 : 실행중인 자바(JVM/메모리) 데이터를 외부(C:드라이브)에 저장하는 목적
		* 저장하는 방법 다양 : DB vs 파일처리 vs 톰캣세션 vs 브라우저 세션(sessionStorage, localStorage) vs 자바 JVM
	3. 클래스

[ 서블릿 파일 클래스 ]
	1. [ 업로드 라이브러리 ]
		cos.jar , commons(Spring)
	2. commons 라이브러리 준비
	3. 톰캣10.0 이후 버전에서는 commons.jar 사용하기 위해서는 마이그레이션 필요하다.
